How to use GamePanel?

First, you need to install all utilities with the GamePanel CLI that can be found here: https://github.com/Crafii9YT/gamepanel Then you need to open the downloaded GamePanel.py! Next, you need to create the GamePanel directory by executing the command "createdir GamePanel". Now you can install the core app, by first going into the GamePanel directory, by executing "gotodir GamePanel", and then executing "install core". The files where created. Now install the GamePanel_Dash.py! You can now execute it, any visit the panel with your local IP, or on an Server with the server IP. To log into your admin account, you need to go into your users folder. On Windows: documents/GamePanel/users/users.json On Linux: (The selected directory)
